# HumiDroplet
The IoT powered weather station

## Components
  - NodeMCU v0.9 development board (ESP8266)
  - DHT22 (humidity)
  - BMP280 (pressure and temperature)
  - A simple photoresistor (luminance)

## Setup
  | NodeMCU | Sensor pin     |
  |--------:|:---------------|
  | D1      | SCL            |
  | D2      | SDA            |
  | D3      | DHT Data       |
  | D5      | Jumper         |
  | A0      | Photoresistor  |
